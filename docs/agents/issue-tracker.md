# Issue Tracker

Issues are tracked in **GitHub Issues** using the `gh` CLI.

## Configuration

- Repository: `hm999999/mattpocock-skills-setup`
- Uses `gh issue create` for creating issues
- Labels applied via `gh issue edit --label`

## Workflow

1. Run `gh issue create --title "..." --body "..." --label "..."`
2. Issues appear at: https://github.com/hm999999/mattpocock-skills-setup/issues
